About my project

My project is a web application based on JavaScript, Angular.js, ASP.NET, and Microsoft SQL Server technologies. It primarily focuses on employee and employer registration, offering basic CRUD (Create, Read, Update, Delete) operations with inter-table relationships. Throughout the development process, I prioritized functionality over extravagant styling. In the future, this system can accommodate detailed employee and employer profiles, tracking, observations, applications, and more.
